## Installation

Currently, only the Android version is available.

[![Google Play Download][google-play-badge]][google-play-download]

## Usage

```bash
# Install dependencies
$ npm install

# Run Metro bundler
$ npm start

# Run on Android in development
$ npm run android

# Run on IOS in development
$ npm run ios

# Lint
$ npm test
```
